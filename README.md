# turku-ui-varaamo

### install
setup folders like this
 ```
/
  /varaamo
  /turku-ui-varaamo
 
  ```
 and then navigate to /varaamo and install
  ```
 npm install ../turku-ui-varaamo 
  ```
 ### developing
 npm link
 ```
  cd /turku-ui-varaamo
  npm link
  cd /varaamo
  npm link turku-ui-varaamo
   ```
